## 1.3.0

- 将新增的10个风险项（R0136-R0145）补充到BS00（全场景）及BS01-BS14各业务场景中
  - R0136 合成身份欺诈 → RS09 身份识别风险，覆盖10个场景
  - R0137 先买后付(BNPL)欺诈 → RS08 支付/金融风险，覆盖7个场景
  - R0138 礼品卡/充值卡欺诈 → RS08 支付/金融风险，覆盖6个场景
  - R0139 友好欺诈 → RS03 客户风险，覆盖7个场景
  - R0140 会员/订阅滥用 → RS01 营销风险，覆盖7个场景
  - R0141 地理位置欺诈 → RS04 非法请求风险，覆盖6个场景
  - R0142 中间人攻击 → RS10 终端对抗风险，覆盖10个场景
  - R0143 OAuth/SSO授权滥用 → RS09 身份识别风险，覆盖全部15个场景
  - R0144 域名/品牌仿冒 → RS04 非法请求风险，覆盖9个场景
  - R0145 内容农场风险 → RS05 内容风险，覆盖6个场景
- 修正业务场景中4处风险分类错误：
  - R0115（恶意广告投放）从RS06（内部风险）移至RS05（内容风险），影响5个场景
  - R0039（负面舆情）从RS06（内部风险）移至RS02（合规风险），影响14个场景
  - R0062-001（信用卡/借款套现）从RS02（合规风险）移至RS08（支付/金融风险），影响6个场景
  - R0062（非法套现）从RS03（客户风险）移除，仅保留在RS08（支付/金融风险），影响5个场景
